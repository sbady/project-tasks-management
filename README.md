# Project task management

An Obsidian plugin for managing projects, tasks, goals, calendar plans, and focus sessions in a Markdown-first workflow.

## Current status

This repository contains the first public `0.1.0` release of the plugin.

The current product direction focuses on:

- project notes linked to project folders
- task notes as separate Markdown files
- goals for week, month, and quarter
- dashboard-based planning
- calendar and kanban workflows
- focus sessions with Pomodoro support

## Development

```bash
npm ci
npm run typecheck
npm run build
```

## Release files

The release workflow publishes:

- `main.js`
- `manifest.json`
- `styles.css`
